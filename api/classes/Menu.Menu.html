handler()

handler(string|array $names, array $attributes, string $element) : \Menu\MenuHandler

Get a MenuHandler.

This method will retrieve ItemLists by name, If an ItemList doesn't already exist, it will be registered and added to the handler.

// Get the menu handler that handles the default name $handler = Menu::handler(); // Get a named menu handler for a single name $handler = Menu::handler('backend'); // Get a menu handler that handles multiple names $handler = Menu::handler(array('admin', 'sales'));

Parameters

The name this handler should respond to
